# SCOPE Accelerator POC

A Streamlit proof of concept that generates standardized
SCOPE scripts from metadata.

## Features

- Job metadata configuration
- Input and output path validation
- JSON-based schema input
- Filter configuration
- SCOPE script preview
- Downloadable `.scope` script
- Downloadable JSON configuration

## Project files

- `app.py` — Streamlit application
- `requirements.txt` — Python dependency
- `app.yaml` — application startup command
- `.gitignore` — excluded files

## Important

The generated SCOPE code is a conceptual POC template.
Internal syntax, extractors, outputters, and supported libraries
must be validated before production use.
